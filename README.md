# PortalDeNoticias

PortalDeNoticias es una aplicación web desarrollada con **Java y Spring Boot** que permite gestionar y visualizar noticias a través de una plataforma sencilla y estructurada. El sistema está diseñado para administrar publicaciones, mostrar contenido informativo y permitir la interacción con una base de datos para almacenar y recuperar noticias.

Este proyecto forma parte de un ejercicio académico orientado al aprendizaje de desarrollo backend con Java, APIs REST y conexión con bases de datos, aplicando buenas prácticas de programación y arquitectura de software.

La aplicación permite gestionar noticias mediante operaciones como creación, consulta y administración de contenidos, facilitando la organización de información dentro de un portal informativo.

---

# Tecnologías utilizadas

- Java
- Spring Boot
- Maven
- REST API
- Base de datos relacional
- Git y GitHub

---

# Funcionalidades

- Crear noticias
- Consultar noticias
- Listar publicaciones
- Conexión a base de datos
- API REST para gestión de contenido

---

# Estructura del proyecto

El proyecto sigue una arquitectura basada en capas:

- **Controller** → Maneja las peticiones HTTP  
- **Service** → Contiene la lógica de negocio  
- **Repository** → Acceso a la base de datos  
- **Model** → Entidades del sistema
